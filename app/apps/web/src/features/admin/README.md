# Admin Frontend Scope

Admin screens are scoped by the selected Dream Project event.

Core screens:

- Dream dashboard
- Event operation
- Member and participating schools
- Application and selection
- Submission confirmation
- Benefit and support
- Activity certificates
- Review score reports
- Mail and notices
- Requests
- External DB connection settings
- History

Important UI rule:

The selected event name must always be visible on event-scoped screens.
