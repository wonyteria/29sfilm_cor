# API Module Plan

Initial modules:

- auth: admin and teacher login
- events: event and Dream Project operation
- applications: teacher applications and admin selection
- submissions: external DB sync, matching, manual review
- mails: templates, scheduled mails, send logs
- benefits: coupons and snack support
- certificates: activity certificate templates and generated PDFs
- reports: score reports
- requests: teacher requests and admin replies
- audit: admin action logs

The first implementation milestone should cover auth, events, applications, selection, mail templates, and audit logs.
