# 29 WITH Dream Project Test App

29 WITH는 29 Platform의 대외협력 운영을 관리하기 위한 테스트 앱입니다.
현재 버전은 영상 꿈나무 양성 프로젝트, 즉 꿈프의 접수, 선정, 출품 확인, 혜택 지급, 활동확인서, 심사표, 히스토리 보관 흐름을 관리자와 선생님 화면으로 나누어 검증할 수 있게 구성되어 있습니다.

## 바로 열기

- 테스트 앱: `outputs/29with-test-app/index.html`
- GitHub Pages를 켜면 루트 `index.html`이 테스트 앱으로 이동합니다.

## 주요 기능

### 관리자

- 꿈프 행사 등록, 수정, 모집 시작, 종료
- 신청자 관리 및 선착순/패널티/성실 참여 기준 선정
- 쿠폰 엑셀 업로드 및 구독권 지급
- 출품작 엑셀 업로드 기반 학교명/소속명 자동 매칭
- 유사 매칭, 미확인 출품작 확인 요청
- 최종 출품 리스트 승인
- 활동확인서 레퍼런스 업로드 및 PDF 발급 상태 관리
- 심사 결과 반영 및 심사표 공개
- 예약 메일, 템플릿, 관리자 답변, 히스토리 관리

### 선생님

- 신청 가능한 행사 확인 및 꿈프 신청
- 여러 꿈프 동시 참여 현황 확인
- 학생/작품 출품 상태 확인
- 구독권, 간식비, 활동확인서, 심사표 상태 확인
- 활동확인서/심사표 수정 요청
- 공지, 메일, 관리자 답변 확인
- 참여 이력 및 베네핏/패널티 상태 확인

## 검증

로컬에서 아래 명령으로 주요 화면과 흐름을 점검할 수 있습니다.

```bash
node outputs/29with-test-app/validate.mjs
```

검증 항목에는 엑셀 매칭, 쿠폰 지급, 활동확인서 발급, 심사표 공개, 메일/문의/히스토리 흐름이 포함됩니다.
