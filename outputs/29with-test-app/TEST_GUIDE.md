# 29 WITH 테스트 가이드

## 열기

`index.html`을 브라우저에서 열어 테스트합니다.

## 관리자 테스트 순서

1. 왼쪽 상단 역할이 `관리자`인지 확인합니다.
2. 상단 행사 선택에서 테스트할 꿈프를 선택합니다.
3. `꿈프 대시보드`에서 진행 중인 꿈프 카드를 누르고, 아래 상세 패널의 `행사 정보 수정`, `신청/선정`, `출품 확인`, `처리할 일 보기` 버튼을 확인합니다.
4. `새 행사 등록`을 눌러 행사명, 행사 유형, 마감, 목표 작품 수, 공모기간, 총상금, 주제, 포스터 URL, 홈페이지/출품 URL을 입력하고 등록합니다.
5. `행사 운영`에서도 `새 행사 등록` 버튼이 보이는지 확인하고, 선택된 행사 정보를 수정/저장합니다. 여기서 입력한 공모기간/총상금/주제/포스터/안내사항은 선생님 `신청 가능한 행사 > 자세히 보기`에 표시됩니다.
6. `신청/선정`에서 신청자 선정, 쿠폰 지급, 메일 예약 흐름을 확인합니다.
7. `출품 확인`으로 이동합니다.
8. `출품 엑셀 분석` 버튼을 누릅니다.
   - 파일을 선택하지 않으면 샘플 데이터로 분석됩니다.
   - 직접 테스트하려면 `sample-submissions-29sfilm.csv` 또는 `sample-submissions-shortform.csv`를 업로드합니다.
9. 결과를 확인합니다.
   - 학교명과 엑셀 `소속`이 완전히 같으면 `자동확인`
   - 띄어쓰기나 한 글자 차이처럼 비슷하면 `확인 필요`
   - 후보가 없으면 `미확인`
10. 학교 행을 열어 작품별 상태를 확인하고, 필요한 경우 `수동 매칭`을 누릅니다.
11. `확인 필요`가 남아 있지 않을 때 `최종 승인`을 누릅니다.
12. `활동확인서`, `심사 결과`, `메일/공지`, `혜택/지원`에서 후속 흐름을 확인합니다.

## 선생님 테스트 순서

1. 왼쪽 상단 역할을 `선생님`으로 전환합니다.
2. `내 대시보드`에서 신청 상태, 출품 확인 상태, 혜택, 문서 상태를 봅니다.
3. `신청 가능한 행사`에서 행사 카드의 공모기간/주제/총상금을 확인하고 `자세히 보기`에서 포스터, 안내사항, 홈페이지/출품 URL까지 확인합니다.
4. 진행 중인 꿈프가 여러 개일 때 상단 행사 선택 또는 프로젝트 전환 버튼으로 각각 분리되어 보이는지 확인합니다.
5. `학생/작품 관리`에서 미확인 작품에 직접 작품 정보를 입력해 봅니다.
6. `내 프로필`에서 기본 정보가 읽기 전용으로 보이는지 확인하고, `수정`을 눌렀을 때 연락처/선생님 증빙자료를 수정 후 저장합니다.
7. `학교명 변경 요청`을 눌러 기존 학교명, 변경 희망 학교명, 변경 사유를 입력하고 요청합니다.
8. `활동확인서/심사표`에서 작품별 문서 준비 상태 리스트를 확인하고, 작품별 `자세히`와 `수정 요청` 팝업을 테스트합니다.
9. `문의하기`를 눌러 문의 내용을 입력하고 관리자 요청함에 전달되는지 확인합니다.
10. `공지/메일함`의 `중요 알림`에서 활동확인서 발급, 심사표 공개, 관리자 답변, 출품 확인 필요 알림을 확인합니다.
11. `혜택/지원`에서 관리자 처리 결과가 선생님 화면에 어떻게 보이는지 확인합니다.

## 실제 엑셀 업로드 기준

- 29초영화제: `소속`, `작품제목`, `감독`, `예심평점`, `수상결과` 컬럼을 사용합니다.
- 29역숏폼왕: `소속`, `제목`, `출품자`, `원본URL`, `평점` 컬럼을 사용합니다.
- `.xlsx` 파일은 브라우저에서 SheetJS가 로드되어야 읽을 수 있습니다.
- `.xls`가 HTML 테이블 형식이거나 `.csv` 파일이면 기본 파서로 읽습니다.
