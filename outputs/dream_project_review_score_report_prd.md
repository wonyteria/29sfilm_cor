# PRD - 29 Platform
## Dream Project(꿈나무양성프로젝트) - 심사 결과 반영 및 심사표 PDF 다운로드 프로세스

---

## 1. 프로젝트 개요

본 문서는 29초영화제 또는 29역숏폼왕의 수상작 발표 이후, 꿈나무양성프로젝트 출품작 리스트에 심사 결과를 자동 반영하고 담당자와 관리자가 웹에서 확인하거나 PDF로 다운로드할 수 있도록 하는 기능을 정의한다.

기존에는 별도의 심사표를 수동 제작하여 담당자에게 전달했으나, 웹페이지 완성 후에는 외부 관리자 페이지의 예심 점수 및 본심 결과 값을 가져와 출품작 리스트에 자동 반영한다.

핵심 목적:

- 29초영화제/29역숏폼왕 관리자 페이지에서 심사 결과 데이터 수집
- 꿈프 담당자별 출품작 리스트에 순위 및 본심 진출 여부 표시
- 수상작 발표 이후 심사 결과 반영 완료 메일 자동 발송
- 담당자와 관리자가 심사 결과 포함 출품작 리스트 확인
- 심사표 PDF 다운로드 제공

---

## 2. 사용자 유형

### 2.1 관리자

권한:

- 행사별 심사 결과 동기화
- 담당자별 심사 결과 리스트 확인
- 순위 산정 결과 확인
- 본심 진출 여부 확인
- 심사표 PDF 생성 및 다운로드
- 심사 결과 반영 안내 메일 발송 및 재발송

### 2.2 꿈나무양성프로젝트 담당자

권한:

- 본인 학교/팀의 출품작 리스트 확인
- 출품작별 순위 확인
- 29초영화제 본심 진출 여부 확인
- 심사표 PDF 다운로드
- 심사 결과 반영 안내 메일 수신

## 3. 적용 대상

본 기능은 아래 조건을 만족하는 신청 건에 적용한다.

- 꿈나무양성프로젝트 신청 상태가 `선정` 또는 `출품완료`
- 출품작 리스트가 생성된 상태
- 관리자 최종 출품작 리스트 승인 완료
- 수상작 발표 이후 심사 결과 값이 외부 관리자 페이지에 반영된 상태

정책:

- 심사 결과는 수상작 발표 이후에만 담당자에게 노출한다.
- 수상작 발표 전에는 관리자만 동기화 준비 상태를 확인할 수 있다.
- 심사 결과 반영 가능 시점은 관리자가 행사별로 설정한다.

### 4.1 29초영화제

출처:

- 29초영화제 관리자 페이지
- 출품작 관리 페이지
- 심사 관리 페이지

필수 수집 정보:

- 외부 행사 ID
- 작품 고유 ID
- 소속명

- 감독명
- 예심 평점
- 수상결과
- 작품 URL

본심 진출 판단:

- 29초영화제 관리자 페이지의 `수상결과` 값이 `1차통과`인 경우 본심 `진출`로 표시한다.
- `1차통과`가 아닌 경우 본심 `-` 또는 `미진출`로 표시한다.

### 4.2 29역숏폼왕

출처:

- 29역숏폼왕 관리자 페이지

필수 수집 정보:

- 외부 행사 ID
- 작품 고유 ID

- 평점

## 5. 출품작 리스트 열 구성

심사 결과 반영 후 담당자와 관리자 화면의 출품작 리스트에는 아래 열을 표시한다.

공통 열:

- 소속/팀명
- 작품명
- 감독명
- 활동확인서
- 순위

29초영화제 전용 열:

- 본심

열 표시 예시:

```text
소속/팀명 | 작품명 | 감독명 | 활동확인서 | 순위 | 본심
```

정책:

- 29초영화제는 `본심` 열을 표시한다.
- 29역숏폼왕은 `본심` 열을 숨기거나 `해당 없음`으로 표시한다.
- 활동확인서 열에는 활동확인서 PDF 다운로드 버튼 또는 상태를 표시한다.
- 순위와 본심 값은 수상작 발표 이후 심사 결과 반영이 완료된 뒤 표시한다.

---

## 6. 순위 산정 기준

순위는 같은 소속/팀명에 매칭된 출품작 리스트 안에서 산정한다.

29초영화제 순위 기준:

- 예심 평점 기준 내림차순 정렬

29역숏폼왕 순위 기준:

- 평점 기준 내림차순 정렬

예시:

```text
서울고등학교 출품작 3편

1위: 작품 A / 예심 평점 4.8
2위: 작품 B / 예심 평점 4.2
3위: 작품 C / 예심 평점 3.9
```

동점 처리:

- 같은 점수인 경우 동일 순위로 표시한다.
- 동일 순위 다음 순위는 건너뛰는 방식으로 표시한다.

예:

```text
1위: 작품 A / 4.8
1위: 작품 B / 4.8
3위: 작품 C / 4.2
```

정책:

- 전체 영화제 순위가 아니라 해당 꿈프 담당자의 소속/팀명 출품작 안에서의 순위다.
- 점수 값이 없는 작품은 순위 산정에서 제외하고 `점수 미등록`으로 표시한다.
- 점수 값이 나중에 추가되면 재동기화 후 순위를 다시 계산한다.

---

## 7. 본심 진출 여부

본심 진출 여부는 29초영화제에만 적용한다.

판단 기준:

- 외부 관리자 페이지의 `수상결과` 값이 `1차통과`이면 `진출`
- 그 외 값이면 `-` 또는 `미진출`
- 값이 비어 있으면 `결과 미등록`

표시값:

- 진출
- 미진출
- 결과 미등록

정책:

- 29역숏폼왕에는 본심 진출 여부를 표시하지 않는다.
- 본심 값은 수상작 발표 이후에만 담당자 화면에 노출한다.
- 관리자는 수집된 원본 수상결과 값을 확인할 수 있다.

## 8. 심사 결과 반영 시점

심사 결과는 수상작 발표 이후 반영한다.

반영 방식:

1. 관리자가 행사별 `심사 결과 반영 시작` 버튼 클릭
2. 시스템이 외부 관리자 페이지에서 점수 및 결과 데이터 동기화
3. 출품작 리스트의 각 작품과 외부 심사 결과 매칭
4. 소속/팀명별 순위 계산
5. 본심 진출 여부 계산
6. 담당자 화면에 심사 결과 표시
7. 심사 결과 반영 완료 메일 발송

정책:

- 수상작 발표 전에는 담당자에게 점수, 순위, 본심 여부를 노출하지 않는다.
- 자동 동기화가 가능하더라도 최종 공개는 관리자가 `심사 결과 반영 시작`을 실행한 이후로 제한한다.
- 심사 결과가 반영되면 담당자에게 별도 안내 메일을 발송한다.

---

## 9. 심사 결과 매칭 기준

심사 결과는 기존 출품작 리스트와 외부 관리자 페이지의 작품 데이터를 매칭하여 반영한다.

기본 매칭 기준:

- 행사 ID 일치
- 작품 고유 ID 일치
- 작품 URL 일치

보조 매칭 기준:

- 소속명 또는 팀명 일치
- 작품명 일치
- 감독명 일치

정책:

- 작품 고유 ID 또는 작품 URL이 있으면 이를 우선 사용한다.
- 작품명만으로 자동 매칭하지 않는다.
- 보조 기준만 일치하는 경우 `수동확인필요` 상태로 표시한다.
- 관리자는 수동으로 심사 결과를 출품작에 연결할 수 있다.

## 10. 담당자 페이지

심사 결과 반영 이후 담당자 마이페이지의 출품작 리스트에 순위와 본심 정보를 표시한다.

표시 정보:

- 행사명
- 학교명
- 소속/팀명
- 작품명
- 감독명
- 활동확인서 다운로드
- 순위
- 본심 진출 여부
- 심사표 PDF 다운로드

상태 문구:

심사 결과 반영 전:

```text
수상작 발표 이후 심사 결과가 반영됩니다.
```

심사 결과 반영 후:

```text
심사 결과가 반영되었습니다. 심사표 PDF를 다운로드할 수 있습니다.
```

정책:

- 담당자는 본인 학교/팀의 결과만 확인할 수 있다.
- 담당자는 원본 예심 평점/평점 숫자를 볼 수 없고, 순위와 본심 진출 여부만 확인한다.
- 관리자는 원본 점수도 확인할 수 있다.

관리자 액션:

## 12. 심사표 PDF

심사표는 담당자별 출품작 리스트를 PDF로 생성한다.

PDF 포함 항목:

- 행사명
- 학교명
- 담당교사명
- 소속/팀명
- 작품명
- 감독명
- 순위
- 본심 진출 여부
- 생성일

정책:

- 29초영화제 심사표에는 본심 진출 여부를 포함한다.
- 29역숏폼왕 심사표에는 본심 열을 제외한다.
- 원본 예심 평점/평점은 담당자용 PDF에 포함하지 않는다.
- 관리자용 PDF에는 설정에 따라 원본 점수를 포함할 수 있다.
- PDF는 심사 결과 반영 이후 다운로드 가능하다.

파일명 예시:

```text
2026_제13회박카스29초영화제_서울고등학교_심사표.pdf
```

## 13. 심사 결과 반영 완료 메일

심사 결과가 반영되면 담당자에게 별도 안내 메일을 발송한다.

메일 발송 시점:

- 관리자가 심사 결과 반영을 완료한 직후

메일 제목 예시:

```text
[29 Platform] 영상 꿈나무 양성 프로젝트 심사 결과 반영 안내
```

메일 포함 내용:

- 담당자명
- 학교명
- 행사명
- 심사 결과 반영 완료 안내
- 출품작 리스트에서 순위 및 본심 진출 여부를 확인할 수 있다는 안내
- 심사표 PDF 다운로드 가능 안내
- 담당자 페이지 바로가기

메일 문구 예시:

```text
안녕하세요. 29초영화제사무국입니다.

{행사명} 영상 꿈나무 양성 프로젝트 출품작의 심사 결과가 반영되었습니다.

담당자 페이지에서 출품작별 순위와 본심 진출 여부를 확인하실 수 있으며,
심사표 PDF를 다운로드하실 수 있습니다.
```

정책:

- 심사 결과 반영 완료 메일은 담당자별 1회 자동 발송한다.
- 관리자는 메일을 재발송할 수 있다.
- 메일 발송 성공/실패 이력을 저장한다.

## 15. 예외 상황

### 15.1 점수 미등록

- 순위를 계산하지 않는다.
- 담당자 화면에는 `점수 미등록` 또는 `결과 준비중`으로 표시한다.
- 관리자 화면에는 점수 미등록 작품 수를 표시한다.

### 15.2 작품 매칭 실패

- 심사 결과를 자동 반영하지 않는다.
- 관리자 화면에 `수동확인필요`로 표시한다.
- 관리자가 수동 매칭할 수 있다.

### 15.3 수상결과 값이 예상과 다른 경우

- 29초영화제 수상결과 값이 `1차통과` 외의 신규 값이면 원본 값을 저장한다.
- 본심 여부는 `수동확인필요`로 표시한다.
- 관리자가 본심 진출 여부를 확정할 수 있다.

### 15.4 심사 결과 반영 후 점수 변경

- 재동기화 시 기존 순위를 다시 계산한다.
- 변경 이력을 저장한다.
- 이미 발송된 메일은 자동 재발송하지 않는다.
- 관리자가 필요 시 안내 메일을 재발송한다.

저장 데이터:

- 예심 평점 또는 평점

- 계산된 순위

- 심사표 PDF 재생성

## 17. 개발 원칙

- 담당자에게는 원본 점수보다 순위와 본심 진출 여부 중심으로 보여준다.
- 원본 점수는 관리자 화면에서 확인 가능해야 한다.
- 순위는 전체 행사 기준이 아니라 같은 소속/팀명 출품작 리스트 안에서 계산한다.
- 29초영화제의 본심 여부는 수상결과 `1차통과`를 기준으로 한다.
- 29역숏폼왕은 본심 열을 제공하지 않는다.
- 심사 결과 공개는 수상작 발표 이후 관리자 실행을 기준으로 한다.
- 심사표 PDF와 메일 발송 이력은 별도로 저장한다.
